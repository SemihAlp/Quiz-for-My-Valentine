# Quiz for My Valentine

Çiftler için basit ve tatlı bir bilgi yarışması oyunu.

## Özellikler
- Havalı bir spinner ile 1-25 arası soru oluşturma
- Animasyonlu soru kartları
- Uyumluluk analizi
- Ses efektleri

## Teknolojiler
- Vanilla JavaScript
- HTML5 / CSS3
- Web Audio API
